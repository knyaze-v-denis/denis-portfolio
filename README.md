# Denis Knyazev — Portfolio

Personal portfolio website designed and built from scratch using Next.js.

This project combines product design thinking with front-end implementation and focuses on clarity, motion and responsive layout.

## Features

- Responsive layout
- Light / dark theme with animated transition
- RU / EN localization
- Animated UI reveal
- Project cards with adaptive layout
- Timeline layout for experience and education

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Custom design system

## Project structure

app/ — routes and layout
components/ — UI and sections
components/ui/ — buttons, links, cards, tags
components/motion/ — animation utilities
lib/ — helpers and i18n
messages/ — RU / EN translations
public/ — static assets

## Development

Install dependencies

npm install

Run development server

npm run dev

Build project

npm run build
